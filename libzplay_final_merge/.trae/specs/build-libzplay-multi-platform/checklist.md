# Checklist

## 源码整理验证
- [ ] x86 源代码完整且可独立编译
- [ ] x64 源代码完整且可独立编译
- [ ] CMake 构建系统对 x86 和 x64 都有效
- [ ] github 项目中的所有有价值修改已整合
- [ ] libzplay_x64 editon 的所有修复已验证并整合

## 编译器工具链验证
- [ ] MinGW 多版本工具链配置正确（至少 1 个 x86 和 1 个 x64 版本）
- [ ] LLVM/Clang 工具链配置正确
- [ ] MSVC 工具链配置正确
- [ ] 所有工具链都能被 CMake 正确识别

## 构建产物验证
- [ ] out/ 文件夹结构符合 final_standard.txt 要求
- [ ] 每个编译器至少有一种成功的构建组合
- [ ] x86 和 x64 架构的 DLL 都成功生成
- [ ] release 和 debug 配置都成功生成
- [ ] 至少一种优化级别（-O3 或 -Os）成功生成
- [ ] 动态库（DLL）和导入库都正确生成
- [ ] 头文件正确复制到 include 目录

## 构建产物质量验证
- [ ] x86 DLL 使用 objdump 验证为 i386 架构
- [ ] x64 DLL 使用 objdump 验证为 x86-64 架构
- [ ] 导入库（.a/.lib）与对应 DLL 匹配
- [ ] 示例程序能使用构建产物成功编译
- [ ] 示例程序能正常运行播放音频

## 清理验证
- [ ] github/ 文件夹已删除
- [ ] libzplay_x64 editon/ 文件夹已删除
- [ ] 整理后的源码保存在 out/src_x86/ 和 out/src_x64/
- [ ] example/ 文件夹已更新为使用新构建的库

## 文档验证
- [ ] 构建说明文档完整（BUILD.md 或类似）
- [ ] 使用说明文档完整（README.md）
- [ ] 记录了所有构建选项和组合方式
- [ ] 记录了已知的构建问题和解决方案
